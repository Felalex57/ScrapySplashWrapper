Welcome to ScrapySplashWrapper's documentation!
===============================================

This is a module of `Lookyloo <https://www.lookyloo.eu>`_ that submit a capture to a `splash <https://splash.readthedocs.io/>`_ instance.

Installation
------------

The package is available on PyPi, so you can install it with::

  pip install scrapysplashwrapper
